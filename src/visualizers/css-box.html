Key Concepts:

  • Content: The inner area where text and images appear
  • Padding: Clear space around the content, inside the border
  • Border: A line that goes around the padding and content
  • Margin: Clear space outside the border, separates the element from others
  • content-box: Width/height apply to content only (default)
  • border-box: Width/height include content, padding, and border
